queries_scribe_apps/
====================

`View code on Github <https://github.com/scribe-org/Scribe-Data/tree/main/src/scribe_data/wikidata/queries_scribe_apps>`_

This directory contains the language data queries for Scribe-Data that query the data that's needed for Scribe applications. The structure is broken down by language, with each language sub-directory then including directories for nouns, prepositions and verbs as needed. Within these data type directories are :code:`query_DATA_TYPE.sparql` SPARQL files.

Use the :code:`View code on GitHub` link above to view the directory and explore the process!
